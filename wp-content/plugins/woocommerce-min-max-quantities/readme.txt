Define minimum/maximum allowed quantities for products, variations and orders.

== Description ==

Min/Max Quantities makes it possible to:

- Define minimum and maximum quantity limits per product/variation, per product category or per order.
- Require specific products, or product categories, to be purchased in predefined multiples.

Looking for help? Read the full documentation [here](https://woocommerce.com/document/minmax-quantities/).
