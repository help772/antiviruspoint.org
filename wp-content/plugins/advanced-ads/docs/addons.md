# Advanced Ads add-ons

Reference list of products and their plugin folder names under `wp-content/plugins/`.

| Add-on                        | Folder name                     | Pricing |
| ----------------------------- | ------------------------------- | ------- |
| Advanced Ads                  | `advanced-ads`                  | free    |
| Advanced Ads AdSense In-feed  | `advanced-ads-adsense-in-feed`  | free    |
| Advanced Ads Browser Language | `advanced-ads-browser-language` | paid    |
| Advanced Ads GAM              | `advanced-ads-gam`              | paid    |
| Advanced Ads Genesis          | `advanced-ads-genesis`          | paid    |
| Advanced Ads Geo              | `advanced-ads-geo`              | paid    |
| Advanced Ads Layer            | `advanced-ads-layer`            | paid    |
| Advanced Ads Page Peel        | `advanced-ads-page-peel`        | paid    |
| Advanced Ads Pro              | `advanced-ads-pro`              | paid    |
| Advanced Ads Responsive       | `advanced-ads-responsive`       | paid    |
| Advanced Ads Slider           | `advanced-ads-slider`           | free    |
| Advanced Ads Sticky Ads       | `advanced-ads-sticky-ads`       | paid    |
| Advanced Ads Tracking         | `advanced-ads-tracking`         | paid    |

## Notes

-   `advanced-ads` is the **base plugin** (free on WordPress.org).
-   **Free add-ons** (per core plugin add-on box): AdSense In-feed and Slider. Slider is described as free for newsletter subscribers.
-   All other add-ons are **paid** (licensed via wpadvancedads.com / EDD).
-   “Page Peel Pro Responsive Sliding Sliders” maps to the four add-on rows above: Page Peel, Pro, Responsive, and Slider.
-   “Sticky Ads and Tracking” maps to Sticky Ads and Tracking (two separate folders).
-   Full path example: `wp-content/plugins/advanced-ads-pro/`
