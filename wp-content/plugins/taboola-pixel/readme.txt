Taboola Pixel is a WordPress plugin that injects the Taboola Pixel code into your website for advanced tracking and analytics.

== Description ==

Taboola Pixel enables you to track user interactions on your WordPress website and integrate with Realize's advertising platform. The plugin provides:

* Easy setup with guided onboarding
* Automatic pixel installation
* Seamless connection to your Realize account

= Features =

* **Simple Setup**: Connect your Realize account with just a few clicks
* **Codeless Solution**: Easy conversion tracking setup using Realize [Codeless solution](https://developers.taboola.com/pixel/docs/codeless-conversions?_gl=1*u4ap9j*_gcl_au*MjA0NTk4NzYzLjE3NTA1ODM0NzM.*_ga*NTE1MjQ3Mjk0LjE3NTAzMzYyNjc.*_ga_ERHENNM7DM*czE3NTE4ODI3MTkkbzU1JGcxJHQxNzUxODgyOTEwJGo1JGwwJGgw)
* **Secure**: Follows WordPress security best practices

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/taboola-pixel` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the Taboola Pixel menu in your WordPress admin.
4. Click "Connect Realize Account" to link your Realize account.
5. Follow the guided setup to install the pixel.

== Frequently Asked Questions ==

= Do I need a Realize account to use this plugin? =

Yes, you need a Realize account to use this plugin. You can sign up at [https://signup.taboola.com/join](https://signup.taboola.com/join)

= Can I track custom events? =

The plugin automatically tracks standard page view events. For custom event tracking it's best to use our [Codeless solution](https://developers.taboola.com/pixel/docs/codeless-conversions?_gl=1*u4ap9j*_gcl_au*MjA0NTk4NzYzLjE3NTA1ODM0NzM.*_ga*NTE1MjQ3Mjk0LjE3NTAzMzYyNjc.*_ga_ERHENNM7DM*czE3NTE4ODI3MTkkbzU1JGcxJHQxNzUxODgyOTEwJGo1JGwwJGgw) or manually modify the code. Read more in our [Dev Center](https://developers.taboola.com/pixel/docs/wordpress-plugin#whats-next)

= How do I know that it works? =

Use our [Testing Tool](https://developers.taboola.com/pixel/docs/testing-tool), available in the Realize platform, to verify that your setup is good.

= Is this plugin GDPR compliant? =

The plugin respects user privacy settings. Please ensure you have proper consent mechanisms in place according to your local privacy laws.

= How can I report security bugs? =

You can report security bugs through the Patchstack Vulnerability Disclosure Program. The Patchstack team helps validate, triage and handle any security vulnerabilities. [Report a security vulnerability.](https://patchstack.com/database/vdp/dca04388-735b-4286-8106-d86541baeab5)
